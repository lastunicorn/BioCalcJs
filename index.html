BioCalc

A biorhythm calculator made in javascript.

Downloads

Download .zip View on GitHub

Description

BioCalc displays the four primary biorhythm charts of a person based on the birthday.

Current Version

1.3.0

Dependences

  • jQuery - 2.0.3 - Go to home page
  • jQuery UI - 1.10.3 - Go to home page
  • jQuery Cookie - 1.3.1 - Go to GitHub page
  • jQuery Color Picker - Go to home page

Current features

  • Calculates and displays the primary biorhythm charts for one person based on the birthday.
  • Easily scroll the biorhythms left-right by dragging with the mouse or using the mouse wheel.
  • Calculates and displays percentage values of the biorhythms for the X day.
  • Saves the current birthday in a cookie for a later visit of the page.

Future features

  • Display secondary, extra and I-Ching biorhythms of a person.
  • Create an options panel that saves user configuration in cookies.
  • Calculate and display compatibility between two persons.
