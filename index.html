BioCalc

A biorhythm calculator made in javascript.

Downloads

Download .zip View on GitHub

Description

BioCalc knows to display the four primary biorhythm charts of a person based on the birthday.

Current Version

1.2.0

Dependences

jQuery, jQuery UI, jQuery ColorPicker, jQuery Cookie

Future features

  • Display secondary, extra and I-Ching biorhythms of a person.
  • Create an options panel that saves user configuration in cookies.
  • Calculate and display compatibility between two persons.
